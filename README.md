## 概要
- UWP環境でのPhotoフォルダ(カメラロール)に対する画像の一覧，保存，取得を行う

## 利用パッケージ
- [HoloLensModule](https://github.com/akihiro0105/HoloLensModule)

## 内容
- PhotoFileControlSample
    + Photoフォルダの一覧を取得し，先頭画像を表示
    + 表示画像をPhotoフォルダに保存
